# Estudos de JavaScript

Este repositório contém uma série de estudos e projetos práticos em JavaScript, organizados em capítulos progressivos. Cada capítulo explora conceitos fundamentais e avançados da linguagem, com exemplos práticos e aplicações reais.

## Estrutura do Projeto

- **1_intro/**: Capítulo 1 - Criação de um site básico sem framework
  - `index.html`: Página principal do site básico
  - `style.css`: Estilos CSS para a página
  - `img/`: Pasta com imagens utilizadas

## Capítulo 1: Site Básico sem Framework

Este capítulo introdutório foca na criação de uma página web simples utilizando apenas HTML e CSS, sem o uso de frameworks ou bibliotecas JavaScript. É ideal para entender os fundamentos da estruturação de páginas web antes de mergulhar em JavaScript.

### Como executar:
1. Navegue até a pasta `1_intro/`
2. Abra o arquivo `index.html` em qualquer navegador web moderno

### Tecnologias utilizadas:
- HTML5
- CSS3
- shahs/
